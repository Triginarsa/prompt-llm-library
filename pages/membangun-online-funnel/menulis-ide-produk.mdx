# Menulis Ide Produk
Menulis ide produk adalah proses mengembangkan dan mendokumentasikan gagasan atau konsep produk yang dapat menjadi dasar untuk pengembangan produk baru.

### Fill In The Blank Prompts

```json copy
Beritahu saya [angka] ide produk yang memungkinkan untuk [jenis bisnis]
```

```json copy
Buatlah daftar 4 ide produk yang bisa saya buat sebagai [jenis bisnis].
```

```json copy
Customer saya kesulitan dengan [pain point] dan mereka ingin [impian]. brainstorm 10 ide produk yang bisa saya buat untuk menyelesaikan masalah mereka.
```

### Open-ended Prompts
```json copy
Menurutmu, apa yang akan menjadi tren terbesar di [industri] dalam 5 tahun ke depan?
```
```json copy
Apa lima tren teratas yang mempengaruhi [jenis industri] saat ini?
```
```json copy
Produk baru apa yang bisa ditawarkan sebagai [jenis industri] yang melayani [niche]?
```
```json copy
Beritahu saya kemana arah [nama industri] ini akan menuju.
```
```json copy
Produk baru apa yang bisa ditawarkan oleh [jenis bisnis] kepada pelanggannya?
```
```json copy
Berikan saya 3 ide produk yang memungkinkan untuk perusahaan konsultan yang membantu pemilik restoran meningkatkan penjualan.
```
```json copy
Beri saya daftar 4 ide produk yang dapat saya buat sebagai bisnis jasa bersih karpet
```
