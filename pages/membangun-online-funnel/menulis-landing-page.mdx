# Menulis Landing Page
Menulis landing page adalah tentang membuat halaman web yang dirancang khusus untuk mengonversi pengunjung menjadi pelanggan atau melakukan tindakan tertentu, seperti mengisi formulir, berlangganan, atau membeli produk atau layanan.

### Fill In The Blank Prompts

```json copy
Tulis landing page yang menawarkan [lead magnet] kepada [audience]. [ebook/webinar/serial video] gratis ini akan membantu mereka:
-[Benefit 1]
-[Benefit 2]
-[Benefit 3]
Mention bahwa ini benar-benar gratis dan bahwa saya dapat membantu mereka karena [include credentials].
```

```json copy
Bisakah Anda membantu saya membuat halaman arahan untuk [produk / layanan] saya yang mengubah pengunjung menjadi pelanggan? Berikut adalah beberapa detail tentang [produk / layanan] saya: [masukkan detail seperti manfaat, poin penjualan unik, audiens target, dll.].
```



### Open-ended Prompts
```json copy
Saya memerlukan landing page yang dapat membantu saya menghasilkan lebih banyak prospek untuk bisnis saya. Bisakah Anda membantu saya membuat halaman yang menyoroti fitur dan manfaat utama dari produk / layanan saya?
```
```json copy
Saya sedang launching produk baru dan saya butuh landing page yang bisa menarik perhatian dari customer potensial. bisakah kamu membuat satu landing page yang menarik secara visual dan mudah untuk di navigasikan?
```
```json copy
Saya ingin mempromosikan acara yang akan datang dan saya memerlukan landing pageyang dapat membantu saya menjual tiket. Bisakah Anda membantu saya membuat halaman yang mencakup semua informasi yang diperlukan dan mendorong pengunjung untuk mendaftar?
```
```json copy
Saya memerlukan landing page yang dapat memamerkan portofolio dan layanan saya sebagai freelancer. Bisakah Anda membantu saya membuat halaman yang menampilkan keterampilan saya dan mendorong klien potensial untuk menghubungi?
```
```json copy
Saya ingin membuat landing page untuk ebook saya dan saya perlu halaman tersebut meyakinkan untuk mengonversi pengunjung menjadi pelanggan. Bisakah Anda membantu saya membuat halaman yang mencakup panggilan tindakan yang jelas dan menampilkan manfaat utama dari buku saya?
```
```json copy
Saya memerlukan landing page yang dapat meyakinkan pengunjung untuk mendaftar untuk uji coba gratis saya. Bisakah Anda membantu saya membuat halaman yang menampilkan fitur unik produk saya dan membujuk pengunjung untuk mengambil tindakan?
```
```json copy
Saya ingin menghasilkan lebih banyak penjualan untuk toko e-commerce saya dan saya memerlukan landing page yang dapat membantu saya mengonversi pengunjung menjadi pelanggan. Bisakah Anda membantu saya membuat halaman yang menarik secara visual dan mencakup deskripsi produk yang meyakinkan?
```
